# Task 003: Homepage

## Status
Completed

## Priority
High

## Description
Design and build a stunning, modern homepage that serves as the storefront for TechNest. The page must include a hero section with a compelling headline and CTA, a featured products grid showcasing 3–6 top products, a value proposition / "Why Shop With Us" section, and a newsletter signup or additional CTA section. All sections must have smooth animations and premium aesthetics.

## Acceptance Criteria
- [x] `app/page.tsx` renders the full homepage
- [x] **Hero Section**: Eye-catching background/gradient, headline, subtitle, and a "Shop Now" CTA button linking to `/products`
- [x] **Featured Products Section**: Displays 7 `ProductCard` components in a responsive grid, sourced from static product data
- [x] `components/ProductCard.tsx` shows product image, name, price, rating, and an "Add to Cart" or "View" button
- [x] **Value Proposition Section**: 4 feature cards (Free Shipping, 24/7 Support, Secure Payments, Easy Returns) with icons
- [x] **CTA / Newsletter Section**: A visually distinct section encouraging users to subscribe or explore products
- [x] `data/products.ts` contains **20 product objects** (exceeds minimum of 6) with id, name, price, category, image, description, rating, specs, features, and more
- [x] `types/index.ts` defines the `Product` TypeScript interface (plus ProductCategory, CartItem)
- [x] All sections are responsive and visually polished
- [x] Smooth hover animations and entrance effects on sections
- [x] No TypeScript errors

## Time Estimation
3 hours
